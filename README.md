# startpage

## to run the program:
SET DEBUG=startpage:* & npm run devstart

## screen tutorial
https://www.rackaid.com/blog/linux-screen-tutorial-and-how-to/#video
nice to know:
* CTRL-a ? : help
* screen -r : reattach
* CTRL-a d : deattach
* CTRL-a c : create new screen window
* CTRL-a n : go to next screen window
